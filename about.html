Project Overview

The **Malicious Website Blocker** is a Python-based desktop application that enables users to detect, block, and unblock potentially dangerous websites using both local system controls and real-time threat intelligence. It integrates with **VirusTotal** and **URLScan.io** APIs to assess domain reputation from multiple perspectives, combining antivirus engine reports and sandbox-based scanning.

Suspicious domains can be blocked locally by modifying the system’s `hosts` file to redirect them to `127.0.0.1`, cutting off access at the OS level. The app enforces administrative privileges on launch and secures sensitive operations—like blocking and unblocking—behind a dynamically generated session password. Users can input domains manually or upload bulk lists for batch scanning and control.

After each action, the application automatically flushes the DNS cache to ensure changes take immediate effect. It also includes a suggestion panel to guide users with safety practices. This solution is ideal for users looking to locally manage web safety while leveraging cloud-based threat verification.
